# DISARM
Colección de ficheros de ejemplo para convertir flujos de AFB (ATT&amp;CK Flow Builder) en JSON de DISARM  


1. Convert_afb_to_DISARM.py (programa que convierte ficheros afb en json)
2. DISARM_Tactics.csv (tabla maestra con las tácticas de DISARM)
3. DISARM_Techniques.csv (tabla maestra con las técnicas de DISARM)
4. Full_DISARM_Techniques.csv (tabla maestra completa para futuros usos de la aplicación)

Se incluyen los siguientes ficheros de ejemplo para hacer una prueba de concepto  

5. Doppelanger_Manual.json (Layer de DISARM la operacion Doppelanger generado a mano en el Navigator. Sirve como contraste o referencia del funcxionamiento del programa)
6. json
7. afb
